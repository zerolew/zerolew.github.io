Keras自定义层传参

发表于 2018-05-25

Keras自定义层的传参方式主要有三种：

  • 在其call方法中传入Input类型的参数
  • 在其call方法中传入list类型的参数
  • 在其__init__方法中传入参数

说明：

  • 传入Input时会出现Layer ... was passed non-serializable keyword arguments的提示，这意味着如果你使用save_model方法或使用ModelCheckpoint保存模型后，无法直接调用load_model方法加载模型。
  • 如果你已经通过前面两种方法保存了模型，又不想重新训练模型。最简单的解决方案是加载保存的模型参数、训练配置、优化器状态等，然后重新编译模型。关于如何实现各种信息的加载，我们可以参考原始load_model方法的实现，然后改写其部分内容实现自己的load_model方法。
