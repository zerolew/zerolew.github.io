Windows7 x64离线安装keras和theano

发表于 2018-05-15 | 分类于 Deep Learning

工具

  • visual studio 2015
  • cuda_9.0.176_windows
  • cudnn-9.0-windows7-x64-v7.1
  • pycharm-community-2018.1.1
  • Miniconda3-latest-Windows-x86_64
  • 391.35-desktop-win8-win7-64bit-international-whql

注: 显卡驱动需要根据显卡型号去NVIDIA官网下载

相关环境

  • 编译环境
    1. 安装visual studio 2015
  • 驱动及CUDA环境
    1. 安装391.35-desktop-win8-win7-64bit-international-whql
    2. 安装cuda_9.0.176_windows，默认会安装到C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v9.0\
    3. 解压cudnn-9.0-windows7-x64-v7.1，将解压的三个文件夹(bin，include，lib)拷贝到CUDA的安装路径下(该路径下存在同名文件夹，合并即可)
  • Python环境
    1. 安装pycharm-community-2018.1.1
    2. 安装Miniconda3-latest-Windows-x86_64(安装路径中不要有空格)

离线环境搭建

方法

  • 方法一: 在线搭建好虚拟环境，将安装好的虚拟环境文件夹(如D:\Miniconda3\envs\tensorflow)拷贝到离线机的对应文件夹中
  • 方法二: 在联网机上获取离线安装包，拷贝到离线环境中进行安装

注:

  • 方法一需要联网机和离线机的配置及相关环境类似
