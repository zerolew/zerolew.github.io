Ubuntu 16.04科学上网

发表于 2018-06-16 | 分类于 Ubuntu

本文假设读者已获得服务器相关信息（服务器ip地址、端口号、用户名、密码、加密方法等），若读者没有以上信息，可以到这里找人合租服务器，合租者会教你如何操作。当然，读者也可以按照这里的教程搭建属于自己的服务器。本文则主要介绍Ubuntu 16.04下Shadowsocks的安装与配置，以方便大家科学上网。

配置

  1. 新建配置文件

  2. 写入配置信息

  3. 修改配置文件”/etc/polipo/config”(先备份，后修改)，并重启Polipo

  4. 修改系统代理 “All Settings -> Network -> Network proxy”，设置为手动方法，ip全部设为”127.0.0.1”，端口号设为”8123”（同”/etc/polipo/config”中的”proxyPort”值）

  5. 启动Shadowsocks，开启你的新世界

  6. 添加启动项，填入相关内容

    注：

    1. 按照自己的情况配置启动项文件中的路径
    2. 启动项中的图标本身不存在，需要另外下载

  7. 浏览器配置（先将系统代理设置为None）

    1. 将系统代理设置为None，后续会配置Chrome浏览器使其自动切换代理模式（直连或使用代理）
    2. 给chrome安装SwitchyOmega插件，下载地址在这里
    3. 切换到SwitchyOmega配置页面，在左侧点击”New Profile…”新建Profile文件
    4. 配置代理服务器的ip地址和端口号（此处端口号为1080）

    5. 在”auto switch”页面中添加”Rule List”并配置”Switch rules”，图中Rule List URL为

    6. 在浏览器右上角点击SwitchyOmega图标，选择自动切换，然后打开google.com试试
