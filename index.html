Ubuntu 16.04安装及配置

发表于 2017-10-01 | 分类于 ubuntu

本文主要介绍Ubuntu 16.04系统的U盘启动盘制作、系统安装、系统个性化设置、常用软件安装及不常用软件清理等过程。

安装

工具

  1. UltraISO
  2. EasyBCD
  3. FixMbr
  4. Ubuntu镜像(.iso)

U盘启动盘制作

  1. 启动UltraISO，”文件”-> “打开”，找到Ubuntu镜像，打开
  2. “启动”->”写入硬盘镜像”
  3. “写入方式”选择”USB-HDD+”
  4. “便捷启动”->”写入新的驱动器引导扇区”->”Syslinux”
  5. 点击”写入”，等待写入完成

磁盘分配

  1. 右键: “计算机”->”管理”->”硬盘管理”
  2. 选择某分区，右键”删除卷”，得到空闲空间(free space)
    注: “删除卷”内的数据会丢失
