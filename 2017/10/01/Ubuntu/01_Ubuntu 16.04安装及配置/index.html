Ubuntu 16.04安装及配置

发表于 2017-10-01 | 分类于 Ubuntu

本文主要介绍Ubuntu 16.04系统的U盘启动盘制作、系统安装、系统个性化设置、常用软件安装及不常用软件清理等过程。

安装

工具

  1. UltraISO
  2. EasyBCD
  3. FixMbr
  4. Ubuntu镜像(.iso)

U盘启动盘制作

  1. 启动UltraISO，”文件”-> “打开”，找到Ubuntu镜像，打开
  2. “启动”->”写入硬盘镜像”
  3. “写入方式”选择”USB-HDD+”
  4. “便捷启动”->”写入新的驱动器引导扇区”->”Syslinux”
  5. 点击”写入”，等待写入完成

磁盘分配

  1. 右键: “计算机”->”管理”->”硬盘管理”
  2. 选择某分区，右键”删除卷”，得到空闲空间(free space)

注: “删除卷”内的数据会丢失

步骤

  1. 插上U盘启动盘
  2. 重启Windows 7
  3. 设置BIOS，从U盘启动(自行百度)
  4. 联网设置: 非联网(建议,拔出网线)
  5. 第三方软件: 安装(建议)
  6. 安装类型: 其他选项
  7. 分区(点击”空闲空间(free space)”，”+”用于分配)
  8. 选择时区、键盘布局；设置用户名、计算机名、密码等
  9. 重启
用途 大小 类型
/boot 250M(建议) 逻辑分区
swap 内存1-2倍 逻辑分区
/ >= 50G(按需分配) 主分区
home 剩余空间 逻辑分区

修改系统引导项(选做)

  1. 进入Windows 7，启动EasyBCD
  2. “添加新条目”->”Linux/BSD”
  3. “类型”选择”GRUB 2”，驱动器选择/boot分区对应的驱动器，添加条目
  4. “编辑引导菜单”，调整顺序及等待时间

  5. 下载FixMbr.exe放至系统盘(Win7)根目录下，管理员身份运行cmd，命令行输入

  6. 重启电脑
